iRefresher

Project for the IoT A.Y. 2021/2022 at the University "La Sapienza". The project is an implementation of an air nebulizer that can be used in every work environment. If the temperature/humidity level go in a negative range the system will react activating the water pump that will be directly connected to a water nebulizer.
